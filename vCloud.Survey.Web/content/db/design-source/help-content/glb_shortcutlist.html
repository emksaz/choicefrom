快捷键清单

我们为您设计了完整的快捷键功能，熟练使用快捷键能大大提高创建问卷的速度。

Windows系统：

  • 放大问卷：Ctrl + ’+’
  • 缩小问卷：Ctrl + ’-’

  • 拷贝节点：Ctrl + ’C’
  • 粘贴节点：Ctrl + ’V’
  • 复制节点：Ctrl + ’D’
  • 剪切节点：Ctrl + ’X’
  • 删除节点：Delete
  • 断开节点所有路径：Ctrl + ’B’
  • 选择所有节点：Ctrl + ’A’

  • 保存问卷：Ctrl + ’S’
  • 保存问卷中间版本：Ctrl + Alt + ’S’
  • Undo：Ctrl + ’Z’
  • Redo：Ctrl + ’Y’

  • 框选节点：Ctrl + 拖曳鼠标框选节点
  • 选择多个节点：Ctrl + 鼠标点选节点

Mac系统：

  • 放大问卷：Cmd + ’+’
  • 缩小问卷：Cmd + ’-’

  • 拷贝节点：Cmd + ’C’
  • 粘贴节点：Cmd + ’V’
  • 复制节点：Cmd + ’D’
  • 剪切节点：Cmd + ’X’
  • 删除节点：Backspace
  • 断开节点所有路径：Cmd + ’B’
  • 选择所有节点：Cmd + ’A’

  • 保存问卷：Cmd + ’S’
  • 保存问卷中间版本：Cmd + Alt + ’S’
  • Undo：Cmd + ’Z’
  • Redo：Cmd + ’Y’

  • 框选节点：Cmd + 拖曳鼠标框选节点
  • 选择多个节点：Cmd + 鼠标点选节点
